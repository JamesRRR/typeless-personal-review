# typeless-personal-review

一键安装 **Personal Review** —— 把 [Typeless](https://typeless.now/) 本地语音记录变成图示化个人复盘的 [Claude Code](https://claude.com/claude-code) Skill。

```bash
npx typeless-personal-review
```

装完在 Claude Code 里说一句：

```
/personal-review          # 最近一周
/personal-review all      # 全量复盘
```

或者直接说「帮我做一下这周的 personal review」。

全程在你自己电脑上跑，语音数据不上传，报告私有。

- 源码 & 说明：https://github.com/JamesRRR/typeless-personal-review
- 前置：Typeless 桌面版（有一些语音记录）· Node 16+ · Python 3
