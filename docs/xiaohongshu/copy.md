## 正文

最近发现一件事：Typeless（那个按住说话就自动转文字的 App）会把你**所有的语音口述永久存在本地**。

我每天对着 AI 口述指令、聊想法、纠结要不要做某件事……积了一两个月，突然想到：这不就是一份最真实的「我到底在想什么」的记录吗？

于是我做了一个 Claude Code 的 Skill —— **Personal Review**。

一句话，它就把我全部的语音记录读一遍，生成一张看得懂自己的复盘图 👇

📊 **它告诉我：**
· 注意力账单——这段时间我的精力到底分给了哪几件事
· 注意力迁移——我的关注点从「亲手做产品」悄悄滑到了「找被动收入」
· 重复问题——我居然反复纠结同一个问题 8 次都没有结论😅
· 未闭环追踪——那些我提过一嘴、然后就忘了的想法，全被它捞了出来
· 还有情绪信号、决策记忆、本周建议……

最戳我的是**注意力迁移图**：它清清楚楚画出我这两个月是怎么从「熬夜写游戏」变成「满脑子小程序矩阵」的。自己都没意识到，被一张图拍在脸上😳

🔒 全程**本地运行**，你的语音数据不出门，报告也是私有的，我不主动分享谁都看不到。

🛠 用法超简单，在 Claude Code 里说一句：
`/personal-review`（最近一周）
`/personal-review all`（全量）
或者直接说「帮我做一下这周的复盘」就行。

已经开源了（MIT），仓库里有一键安装包。**评论区回「复盘」我发你 GitHub 链接** 🔗

你有没有那种「明明很忙、却不知道忙了啥」的感觉？试试让 AI 帮你看一眼吧～
